Overview
This project is a Result Management System developed in the C programming language. It is designed to manage student academic records efficiently using file handling and pointer-based data structures.

The system allows users to perform all essential operations such as adding, viewing, updating, and deleting student records, along with automatic calculation of percentage, grades, and result status.

 Features:
 Add new student records
 View student details
 Update existing records
 Delete student records
 Automatic grade & percentage calculation
 Persistent storage using file handling
 Efficient memory management using pointers
 User-friendly menu-driven interface

Technologies Used:-
Language: C
Concepts:
Pointers
File Handling
Structures
Modular Programming

Sample Functionalities:-
Add student with marks
Calculate percentage automatically
Assign grades based on performance
Store and retrieve records from file
Learning Outcomes:
Practical implementation of file handling in C
Understanding of dynamic memory using pointers
Designing a menu-driven application
Handling real-world data management problems
